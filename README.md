# Nexus
Admin System made by ChatGPT

#Installation
Put this nexus folder in your garry's mod server's garrysmod/lua/autorun folder. The structure should be like this:


Your Garry's Mod Server folder.
└── GarrysModDS

    └── garrysmod
    
        └── lua
        
            └── autorun
            
                └── nexus
                
                    ├── nexus_cl.lua
                    
                    ├── nexus_sv.lua
                    
                    └── plugins
                    
                        └── SamplePlugin
                        
                            ├── plugin_sv.lua
                            
                            └── plugin_cl.lua
